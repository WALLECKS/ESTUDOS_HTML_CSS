CSS INLINE


O CSS inline é adicionado junto ao código HTML por meio do atributo style.
Isso permite atribuir novas propriedades e alterar o visual do elemento.
TEXTO LINKADO COM O CSS.


CSS LINKADO


O css linkado utiliza um outro arquivo que configura toda a pagina do html atraves de suas configuraçoes.
Para isso precisamos criar um outro arquivo.css e depois digitar os seguintes comandos abaixo.

Dentro da tag head digite o codigo ( olhe no arquivo ). E dentro do arquivo css informe as configuraçoes.


Existem alguns valores que podemos atribuir dentro das configuraçoes do css

  1. background-color: ; Define a cor do fundo de um elemento.
  2. background-image: ; Insere uma imagem de fundo na pagina.
  3. background-size: ; Configura o tamanho da imagem no elemento.
    • top: a imagem fica no topo da pagina
    • left: a imagem
  4. backgroud-position:valor1, valor2; Define a posiçao da imagem em relação a caixa.
    • a posiçao nos valores sao vertical e horizontal
  5. backgroud-attachment: Define se a imagem de fundo rolará com a tela.
  6. backgraoun-repeat: Define se a imagem de fundo pode se repetir ou não.

POP UPS

O pop up é uma pagina sobescrita da pagina principal, ela pode ser utilizado
como um link de direcionamento para outras paginas.
O pop-up é um tipo de janela que se abre no navegador ao visitar uma página web ou acessar uma hiperligação específica.
O pop-up é utilizado por autores de sites para abrir alguma informação extra ou como meio de propaganda.
